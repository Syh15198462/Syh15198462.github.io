How to use ShadowSocks to real Internet

发表于 2016-03-26

经过一段时间的学习，发现有些小白连如何使用都不会，鉴于这种情况,我整理出了这篇教程，本文仅适用于小白们，老司机就请绕道啦~

注：在阅读以下内容之前你需要准备一个可用的shadowsocks.json

废话了这么多，就不耽误大家时间了。那么，我们就开始吧！~

Windows

  1. 你需要准备:

    • Shadowsocks(R) for Windows
    • Google Chrome Browser

  2. 食用方法：

    1. 删除你的电脑中360管家&腾讯管家及所有国产杀软
    2. 确认你的代理服务器地址并将它填入代理服务器框中
    3. 确认你的远程连接端口并将它填入远程连接端口框中
    4. 确认你的本地连接端口并将它填入本地连接端口框中（一般默认为1080）
    5. 确认你的密码并将它填入密码框中
    6. 确认你的加密方式并在下拉菜单中选择它
    7. 当然你也可以右键托盘小飞机，选择扫描屏幕上的二维码来快速添加shadowsocks.json
    8. 确认你的协议及混淆选项并在下拉菜单中选择它
    9. 右键托盘小飞机，选择启用系统代理
  3. Enjoy it !(第8步为ShadowsocksR特有选项，Shadowsocks原版用户可跳过)

Android

  1. 你需要准备：

    • Shadowsocks for Android
    • 一个可用的Shadowsocks.json

  2. 食用方法：

    1. 删除Android设备中360管家&腾讯管家及所有国产杀软
    2. 确认你的代理服务器地址并将它填入代理服务器框中
    3. 确认你的远程连接端口并将它填入远程连接端口框中
    4. 确认你的本地连接端口并将它填入本地连接端口框中（一般默认为1080）
    5. 确认你的密码并将它填入密码框中
    6. 确认你的加密方式并在下拉菜单中选择它
    7. 当然你也可以通过扫描二维码，NFC等方式来快速添加shadowsocks.json
    8. 路由选择绕过局域网及大陆地址
    9. IPV6路由：这个看需求，如果你有公网IPV6地址且需要代理就勾上
    10. 分应用代理：看需求。
    11. 绕行模式：即绕过不需代理的Apps，打勾即为不搭理。
    12. UDP转发：若你的服务端支持就打勾 （部分无良ISP会禁大流量UDP，若开启该服务导致无法连接请关闭）
    13. 自动连接：即开机自启
    14. NAT模式：需要ROOT
      （Kcptun为双边加速服务，如果你的服务端部署了该服务则可以填入相应加速地址，GitHub项目地址请看这里）
  3. 全部配置完成后即可点击右上小飞机连接代理服务器，连接后点击”测试网络连通性“检测是否成功出墙。
  4. Enjoy it !
